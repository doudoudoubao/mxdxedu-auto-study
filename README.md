# 慕课自动刷课助手

一个Chrome浏览器扩展，用于自动播放慕课平台视频课程。

## 功能特点

- ✅ 自动播放视频
- ✅ 自动切换下一集
- ✅ 自动检测并跳过已播放视频
- ✅ 静音播放选项
- ✅ 可调节播放倍速 (1x-4x)
- ✅ 防检测机制（模拟人类行为）
- ✅ 自动恢复播放（防止视频暂停/卡顿）
- ✅ 运行状态统计

## 安装步骤

### 方法一：开发者模式安装（推荐）

1. 下载并解压本扩展文件夹

2. 打开Chrome浏览器，地址栏输入：
   ```
   chrome://extensions/
   ```

3. 开启右上角的「开发者模式」

4. 点击「加载已解压的扩展程序」

5. 选择 `chrome-extension` 文件夹

6. 扩展安装完成，浏览器右上角会出现扩展图标

### 方法二：打包安装

1. 在扩展管理页面点击「打包扩展程序」

2. 选择 `chrome-extension` 文件夹

3. 生成 `.crx` 文件后拖入浏览器安装

## 使用方法

1. **登录课程网站**
   - 访问 https://www.mxdxedu.com
   - 登录你的账号
   - 进入需要刷课的课程页面

2. **启动扩展**
   - 点击浏览器右上角的扩展图标
   - 根据需要调整设置：
     - **自动播放**：自动开始播放视频
     - **自动下一集**：视频结束后自动切换
     - **跳过已播放**：自动跳过已观看过的视频
     - **静音播放**：播放时静音
     - **播放倍速**：1-4倍速播放
     - **切换延迟**：切换下一集前的等待时间
   - 点击「开始刷课」按钮

3. **查看状态**
   - 扩展面板显示当前运行状态
   - 显示已完成的集数
   - 显示运行时间

4. **停止运行**
   - 再次点击扩展图标
   - 点击「停止刷课」按钮

## 注意事项

### 关于倍速
- 建议使用 1x-2x 倍速
- 过高的倍速可能被平台检测
- 部分平台会记录实际观看时长

### 关于防检测
- 扩展会模拟鼠标移动和滚动
- 建议不要同时开太多标签页
- 适当休息，避免24小时连续运行

### 关于自动下一集
- 扩展会尝试多种方式查找「下一集」按钮
- 如果自动切换失败，请手动点击
- 切换延迟建议设置 3-5 秒

### 关于跳过已播放
- 扩展会自动记录已观看完成的视频
- 启用「跳过已播放」后，遇到已看过的视频会自动跳过
- 可连续跳过多个已播放视频
- 记录保存在浏览器本地存储中

### 常见问题

**Q: 视频不自动播放？**
A: 浏览器可能阻止了自动播放，请允许该网站的自动播放权限。

**Q: 无法切换下一集？**
A: 可能是页面结构变化，请手动点击下一集按钮。

**Q: 倍速不生效？**
A: 部分播放器会重置倍速设置，请在视频开始后手动调整。

**Q: 被平台检测到？**
A: 降低倍速，增加切换延迟，适当手动操作。
